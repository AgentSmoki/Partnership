## 🔄 Правила пересмотра дорожной карты

1. **Checkpoint** в конце каждого этапа — совместный видеозвонок для обсуждения результатов
2. Если цели этапа не достигнуты на 50% → пересмотр сроков и/или обязанностей
3. Любые изменения в карте фиксируются письменно с датой и подписями обоих партнёров
4. При критических изменениях (pivot, новая ниша, смена технологии) → внеочередной checkpoint
